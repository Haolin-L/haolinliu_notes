## RM65机械臂网络连接

### 查看RM65的IP地址

```bash
cd ws_moveit/src/rm_robot/rm_driver/launch/rm_65_driver.launch
```



```bash
<arg name="Arm_IP"                   default="192.168.21.67"/>    <!-- 设置TCP连接时的IP -->
```



### 网线连接电脑和RM65

>注意检查网线的好坏



### 手动设置IP

>有线连接成功后，打开有线连接的设置，找到IPv4,将方式改为手动

```bash
#地址改为192.168.21.x（保证和机械臂IP在一个网关内，但不可以一样，即不可以为192.168.21.67）
#子网掩码为255.255.255.0
```



### 测试是否成功



```bash
ping 192.168.21.67
#如果能ping通，则一般没问题
roslaunch rm_driver rm_65_driver.launch
#若无报错，则网络连接成功
```



### 常见问题

##### 机械臂的IP无法确定

```bash
#下载wireshark
sudo apt update

sudo apt install wireshark

#启动wireshark
sudo wireshark
#找到eno1,双击，Tell后即机械臂的IP
```
