## 绿联AX300无线网卡问题解决

```bash
#检查模块加载错误
dmesg | tail -20
#预期结果
[  423.996248] Lockdown: modprobe: unsigned module loading is restricted; see man kernel_lockdown.7
[  435.649253] Lockdown: modprobe: unsigned module loading is restricted; see man kernel_lockdown.7
#说明内核锁定了未签名的文件
```





```bash
#检查安全启动状态
mokutil --sb-state
#预期结果
SecureBoot enabled#问题之所在
```



># 解决方法
>
>重启电脑，进入BIOS/UEFI模式，不同电脑进入方式不同，自己查
>
>找到Security/安全设置，选择Secure Boot/安全启动模式，选择disabled/禁用
>
>保存并退出
